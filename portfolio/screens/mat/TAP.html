O que é?

Técnicas Avançadas de Programação referem-se a um conjunto de métodos, práticas e tecnologias modernas utilizadas no desenvolvimento de software para criar sistemas mais eficientes, escaláveis, seguros e de alta performance.
Essas técnicas são essenciais para lidar com a complexidade dos sistemas atuais, que demandam velocidade, confiabilidade e adaptabilidade em um cenário tecnológico em constante evolução.

Objetivos

O objetivo das Técnicas Avançadas de Programação Web e Mobile é desenvolver aplicações altamente eficientes, seguras e escaláveis, garantindo uma experiência de usuário fluida e responsiva em qualquer dispositivo.


Conteudo

Durante todo o Semestre, vimos diversos conteudos e agora por meio deste projeto venho falar um pouco sobre o que estudei ao longo desse semestre.



Vetores e Matrizes


Vetor é uma estrutura de dados unidimensional que armazena uma sequência de elementos do mesmo tipo em uma única linha.
Exemplo: int numeros[5] = {10, 20, 30, 40, 50};

Matriz é uma estrutura bidimensional que armazena dados em linhas e colunas, como uma tabela.
Exemplo: int matriz[2][3] = {{1, 2, 3}, {4, 5, 6}};



CRUD


Durante varios meses fizemos varios Cruds seguindo o modelo MVC, que o Professor disponibilizava, sejam eles N para 1 ou 1 para 1.
Junto com isso aprendemos sobre a utilização da linha de comando do nugget, para criação automatica de tabelas e colunas, para o banco de dados SQL Server.

Apps utilizados

pgAdmin

O pgAdmin é uma ferramenta de código aberto usada para gerenciar e desenvolver bancos de dados PostgreSQL. É uma interface gráfica que permite aos usuários interagir com o banco de dados, executando consultas SQL, visualizando e modificando estruturas de dados e monitorando o desempenho.

Microsoft Visual Studio

Microsoft Visual Studio é um ambiente de desenvolvimento integrado (IDE) da Microsoft para desenvolvimento de software especialmente dedicado ao .NET Framework e às linguagens Visual Basic (VB), C, C++, C# (C Sharp) e F# (F Sharp).

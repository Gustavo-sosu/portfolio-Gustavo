Sobre a Matéria

Professor: Marcelo Tadeu Boer

O que é?

Técnicas Avançadas de Programação referem-se a um conjunto de métodos, práticas e tecnologias modernas utilizadas no desenvolvimento de software para criar sistemas mais eficientes, escaláveis, seguros e de alta performance.
Essas técnicas são essenciais para lidar com a complexidade dos sistemas atuais, que demandam velocidade, confiabilidade e adaptabilidade em um cenário tecnológico em constante evolução.

Objetivos

O objetivo das Técnicas Avançadas de Programação Web e Mobile é desenvolver aplicações altamente eficientes, seguras e escaláveis, garantindo uma experiência de usuário fluida e responsiva em qualquer dispositivo.


Conteudo

Durante todo o Semestre, os alunos com auxilio do professor, ficamos desenvolvendo um site Web, genérico, e que cada um ao final devera personaliza-lo para um tema a escolha.

O que usamos?

Durante as aulas usamos JavaScript explorando a fundo as diversas utilidades dele, css, NPM e o firebase do google

Apps utilizados

Visual Studio Code

O Visual Studio Code (VS Code) é um editor de código-fonte gratuito e versátil, desenvolvido pela Microsoft, que é amplamente utilizado por desenvolvedores de software. Ele é conhecido por sua leveza, flexibilidade e vasta gama de recursos

Firebase

Firebase é uma plataforma de desenvolvimento de aplicativos da Google que facilita a criação, lançamento e gestão de aplicações móveis e web. Oferece uma gama de serviços que abrangem desde autenticação e armazenamento de dados, até notificações e análise de desempenho, proporcionando uma solução completa para desenvolvedores.

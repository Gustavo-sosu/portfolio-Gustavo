SIGO

Sistema de Informatização e Gerenciamento de Oficinas

Projeto Integrador

Imagem de fundo: Designed by Freepik

Resumo da Ideia

Este trabalho propõe um sistema para modernizar a gestão de oficinas mecânicas, com foco em organização, transparência e eficiência. A plataforma visa facilitar o controle de ordens de serviço, veículos, clientes, agendamentos e estoque, além de oferecer um espaço exclusivo para que os clientes acompanhem o histórico de manutenção e serviços em andamento. Com níveis de acesso distintos para administradores, mecânicos e clientes, o sistema promove uma interação segura e prática. Desenvolvido com tecnologias web e mobile, é acessível de qualquer lugar.
A proposta se destaca por atender a um setor ainda marcado por processos manuais, contribuindo para sua profissionalização, melhorando o atendimento ao cliente e auxiliando na tomada de decisões com relatórios e indicadores.

Funcionalidades Principais

  • Cadastro e login de usuários
  • Interface intuitiva e responsiva
  • Controle de ordens de serviço
  • Histórico de atendimentos
  • Gestão de peças

    • O projeto também é pensada para o cliente final, permitindo que ele:

  • Acompanhe o status do seu veículo em tempo real
  • Visualize orçamentos
  • Solicite relatórios detalhados
  • Mantenha um histórico de serviços realizados

O desenvolvimento está sendo realizado em equipe, utilizando metodologias ágeis e ferramentas modernas como GitHub para melhor trabalho em equipe e Visual Studio para codificação.

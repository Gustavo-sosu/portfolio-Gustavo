Sobre a Matéria

Professor: Willian Guimaraes Borges

O que é?

As Estruturas de Dados são formas organizadas de armazenar e gerenciar informações em um sistema computacional, permitindo operações eficientes como inserção, busca, remoção e ordenação.
Elas são fundamentais para otimizar o desempenho de algoritmos e aplicações, impactando diretamente na velocidade e consumo de recursos.

Objetivos

O principal objetivo é garantir que os dados sejam armazenados e acessados da maneira mais eficiente possível, reduzindo tempo de processamento e uso de memória. Isso envolve:
Organização lógica dos dados (como listas, árvores ou grafos).

Conteudo

Durante todo o Semestre, vimos diversos conteudos e agora por meio deste projeto venho falar um pouco sobre o que estudei ao longo desse semestre.




O que é o C

Começamos aprendo um pouco sobre a linguagem C, uma linguagem de programação que se assemelha muito a diversas outras linguagens, sendo uma opção muito utilizada para aprender a programar.

O que foi ensinado?

Durante as aulas, aprendemos sobre os principais conceitos da linguagem C, como variáveis, tipos de dados, operadores, estruturas de controle de fluxo (if, for, while), sobre funções, Vetores, Matrizes, Structs e sobre Pilhas e Filas.

Visual Studio Code

O Visual Studio Code (VS Code) é um editor de código-fonte gratuito e versátil, desenvolvido pela Microsoft, que é amplamente utilizado por desenvolvedores de software. Ele é conhecido por sua leveza, flexibilidade e vasta gama de recursos

OnlineGDB

O OnlineGDB é uma ferramenta online gratuita que permite compilar e depurar código de diversas linguagens de programação, como C, C++, Python, Java, entre outras, sem a necessidade de instalar nenhum software. Ele funciona como um compilador e depurador online, facilitando o desenvolvimento e a resolução de problemas em código.
